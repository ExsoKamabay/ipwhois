``KAMABAY`` WHOIS
--------------------------
**Installation**

.. code :: python

        #windows
        pip install kamabay-ipwhois
        #linux & termux
        pip3 install kamabay-ipwhois
        

``URL`` `https://pypi.org/project/kamabay-ipwhois/`_

**using command line type** ``ipwhois``
